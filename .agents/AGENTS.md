# Pixel-Snap Agent Agency

The Pixel-Snap project is collaboratively built and evolved by a specialized autonomous Agency, acting under the guidance of the Director Agent (Antigravity).

## Agency Roster

- **Sentinel (Repo & Codebase Sentinel)**
  Monitors the repository, codebase architecture, and ensures structural integrity via A2A communication.

- **Weaver (Feature Weaver + Pixel Alchemist)**
  Scaffolds and builds features directly within the Android Kotlin/Compose Codebase. Focuses on bringing the magical UX vision to life.

- **Quality (Quality & Sovereignty Sentinel)**
  Enforces code sovereignty and quality by managing GitHub Actions, ktlint, detekt, and the test suites.

- **Chronicler (Chronicler & Experience Weaver)**
  Handles all documentation, narrative design, UX copy, and chronicles the continuous evolution of the project.

- **Catalyst (Evolution Catalyst & Growth Oracle)**
  Proposes PRs to GitHub, analyzes growth trajectories, and acts as the Oracle managing the human-review loop.

- **Harbinger (Integration Harbinger)**
  Integrates external systems, exposing the MCP Tool Server (capture/analyze/edit/story), the RelicWeave MCP Bus, and monetization (RevenueCat).

- **MemoryWeaver (Memory Weaver)**
  The AI domain expert. Integrates On-Device AI (MediaPipe, ML Kit, local LLM) and curates the Memory Garden Bridge.

- **Critic (Realignment & Alignment Guardian)**
  The immune system of the agency. Reviews proposals, designs, and code to ensure integrity, ethics, sovereignty (HSA alignment), and graceful degradation. Evaluates against 7 dimensions and issues structured, actionable critiques.
